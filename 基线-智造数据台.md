# 基线 · 智造数据台（v1.0）

> **产品**：数据管理 + 基于数据的生产展示（PC 端为主）  
> **状态**：最终需求基线  
> **数据结构**：见 [基线-数据结构.md](./基线-数据结构.md) · 配置 `schemas/智造数据台/presets/jqhc-manufacturing-config.json`

---

## 1. 客户与策略基线

| 项 | 说明 |
|----|------|
| 信息化现状 | 无 ERP/MES/PLM/WMS |
| 制造能力 | 高度自研产线；AGV、智能配方、PLC/传感器 |
| 建设策略 | OT 先行、轻量启动、价值闭环 |
| 首期试点建议 | 药芯焊丝产线（增长亮点、质量风险集中、预算可控） |
| 产品覆盖 | 焊条、实心焊丝、药芯焊丝 |

---

## 2. 产品目标

1. 统一管理数据点定义、采集来源、业务含义与数据质量。
2. 将设备状态、工艺过程、物料流转转化为可查询、可展示、可分析的生产信息。
3. 支撑实时看板、批次追溯、OEE、SPC 与管理 KPI。
4. 全部工序与字段可配置扩展，不写死在代码中。

---

## 3. 用户角色

系统管理员 · 数据工程师 · 自动化工程师 · 工艺工程师 · 质量工程师 · 班组长 · 管理层

---

## 4. 领域模型

### 4.1 数据三大类

- **equipment_status**：启停、模式、转速、电流电压、温度、报警
- **process_parameter**：配方、拉拔力、镀层厚度、填充率、烘干曲线等
- **material_flow**：批次标识、上下线、出入库、AGV 事件

### 4.2 产品工艺（制造域工序链）

见 [基线-数据结构.md §二](./基线-数据结构.md)。

### 4.3 KPI 锚点

质量（一次合格率、Rm、KV₂、A）· 耐蚀（I 值）· 交付（交期、换型、隐性等待）· 效率（OEE）· 能源 · 合规（追溯、碳足迹）

---

## 5. 功能范围（完整定义）

### 5.1 数据治理

| 模块 | 能力 |
|------|------|
| 产线与工序 | 产线登记、工序字典、引用 `template_id` |
| 设备主数据 | 设备 ID、协议连接、所属工序 |
| 数据点管理 | 字段定义、采样、协议映射、规格限、必采标记 |
| 数据字典 | 字段释义、采样逻辑、KPI 关联 |
| 网关配置 | 设备映射、寄存器/OPC UA 节点 |
| 领域配置 | 加载/编辑 `DomainConfig` preset |

### 5.2 数据接入与存储

| 模块 | 能力 |
|------|------|
| 实时接入 | `ProcessSnapshot` / `TelemetryRecord` / `GenericTelemetryPoint` |
| 批次打标 | 关联 product_batch、recipe_id、shift |
| 物料事件 | LINE_ON/OFF、STOCK_IN/OUT、AGV 等 |
| 报警 | 报警字典 + 报警事件 + 处理状态 |
| 归档分层 | hot / warm / cold |

### 5.3 生产展示与分析

| 模块 | 能力 |
|------|------|
| 实时看板 | 工艺趋势、设备状态、当班报警；可配置 Dashboard |
| 历史查询 | 按点/设备/批次/时间范围 |
| OEE | 可用率×表现性×质量率，停机原因 |
| 批次时间轴 | 工序穿透、关键参数摘要 |
| 批次追溯报告 | 上游批次链、追溯断点标识 |
| SPC | 控制图、违规规则、Cpk/Ppk |
| 质量检验 | 检验结果与批次工艺对照 |
| 数据质量监控 | 断流、越界、时钟偏移、缺批次标签 |
| 多产线对比 | 参数与批次对标 |
| 能源 | 单批次/产线能耗、绿电分解 |
| 碳足迹 | 按批次/订单 Scope1/2/3 |
| 管理层 KPI | 一次合格率、交期、OEE、产能利用率等快照 |
| 报表导出 | 追溯报告、OEE、SPC、时序数据 |
| 外部开放 | 查询 API、元数据 API、事件订阅 |

## 6. 关键工艺数据点（摘要）

完整列表以 `jqhc-manufacturing-config.json` 为准。

| 产品 | 关键工序 | 关键 field_id |
|------|----------|---------------|
| 焊条 | 拔丝/切丝/配粉/压涂/烘干 | outlet_diameter_um, length_mm, ingredients, coating_thickness_mm, moisture_pct |
| 实心/药芯 | 粗拔/细拔/镀铜/层绕 | tension_kN, coating_thickness_*_um, spool_id |
| 药芯特有 | 裁带/填充 | strip_thickness_mm, fill_ratio_pct, forming_pressure_MPa |
| 物料 | 包装/入库 | finished_batch_id, agv_id, location |

### 混合均匀度（双算法，可选）

- `mixing_uniformity_method`: `current_fluctuation` | `nir_probe`
- `mixing_uniformity`: 对应算法输出分值
